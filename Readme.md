# 프로젝트 설명
손가락을 통하여 RC카를 제어하는 임베디드 시스템을 설계하였습니다.  
실제 MCU의 data sheet를 읽고 uart , i2c , pwd 등 임베디드시스템에 필요한 통신을 직접 구현하였습니다.
